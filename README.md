# Projeto-Interdisciplinar---GTI
Repositório destinado a arquivar registros do meu projeto da matéria Projeto Interdisciplinar realizado no Instituto Federal do Paraná.

Descrição do Projeto:
O presente projeto neste repositório tem como objetivo criar uma aplicação Web onde o mesmo fará o mapeamento de artistas que moram no município de Pinhais/PR a fim de divulgar seus trabalhos/projetos, conectar com outros artistas, divulgação de eventos para que artistas possam se inscrever e trabalhar em tal evento e suporte por parte do governo disponibilizando recursos (dinheiro, equipamentos e etc) a fim de incentivar o trabalho de tal artista.
